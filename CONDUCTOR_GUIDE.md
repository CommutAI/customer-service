# Conductor App User Guide

## Welcome
This guide will help you use the Conductor App to validate passenger fares and manage QR card scanning during your daily operations.

## Getting Started

### Login
1. Open the Conductor App on your device
2. Enter your staff credentials (username and password)
3. Tap "Login" to access the dashboard

### Dashboard Overview
The main dashboard displays:
- **Current Route/Terminal**: Shows your assigned route
- **Quick Scan Button**: Fast access to QR scanner
- **Recent Validations**: List of recent fare validations
- **Daily Statistics**: Today's transaction count and revenue

## Fare Validation Process

### Step 1: Select Route
- Choose your current terminal/route from the dropdown
- This ensures accurate fare calculation based on distance

### Step 2: Scan QR Card
1. Tap the "Scan QR" button
2. Point your device camera at the passenger's QR card
3. Hold steady until the QR code is recognized
4. The app will automatically capture the card ID

### Step 3: Validate Passenger
After scanning, the app displays:
- **Passenger Name**: Cardholder's full name
- **Passenger Type**: Regular, Student, Senior Citizen, or PWD
- **Registered Destination**: Where the passenger is going
- **Applicable Fare**: Base fare for the route
- **Discount Applied**: 20% for eligible passengers (Student, Senior Citizen, PWD)
- **Final Fare**: Amount to be deducted
- **Current Balance**: Passenger's card balance

### Step 4: Confirm Validation
- Review the fare details
- If correct, tap "Confirm" to deduct fare
- If incorrect, tap "Cancel" to abort

### Step 5: Generate Receipt (Optional)
- After confirmation, you can generate a receipt
- Receipt shows transaction details and remaining balance
- Pass the receipt to the passenger for their records

## Error Handling

### Insufficient Balance
If a passenger's balance is insufficient:
- App will display warning message
- Show required amount vs. available balance
- Advise passenger to top-up before validation
- Cannot proceed with validation until balance is sufficient

### Invalid QR Code
If QR code cannot be scanned:
- Ensure QR code is clearly visible
- Check for proper lighting
- Ask passenger to present card at different angle
- If card is damaged, use manual card ID entry

### Card Not Found
If scanned card ID doesn't exist in system:
- Verify card is registered
- Check if card is from a different terminal
- Contact supervisor for assistance

### Network Issues
If app is offline:
- App enters offline mode
- Validations are queued locally
- Data syncs automatically when connection restores
- Continue operations normally

## Best Practices

### Scanning Tips
1. **Lighting**: Ensure adequate lighting for clear QR code visibility
2. **Distance**: Hold device 6-12 inches from QR code
3. **Angle**: Keep camera perpendicular to QR code surface
4. **Stability**: Hold steady until scan completes
5. **Clean Cards**: Wipe dirty/damaged QR cards before scanning

### Passenger Interaction
1. Greet passengers professionally
2. Explain fare calculation if asked
3. Show receipt details for transparency
4. Handle balance inquiries patiently
5. Report suspicious cards to supervisor

### Security Measures
1. Never share your login credentials
2. Log out when not using the app
3. Report lost/stolen devices immediately
4. Verify card authenticity during scanning
5. Monitor for duplicate or suspicious transactions

## Daily Operations

### Start of Shift
1. Login to the app
2. Select your assigned route/terminal
3. Check daily statistics from previous day
4. Verify device battery and camera functionality

### During Shift
1. Scan each passenger's QR card
2. Validate fares accurately
3. Generate receipts when requested
4. Monitor validation queue in offline mode
5. Handle passenger inquiries professionally

### End of Shift
1. Review daily statistics
2. Check for any pending syncs
3. Log out of the app
4. Report any issues encountered
5. Charge device for next shift

## Troubleshooting

### Camera Not Working
- Check camera permissions in device settings
- Close and reopen the app
- Restart device if issue persists
- Contact technical support if unresolved

### App Crashing
- Force close the app
- Clear app cache if possible
- Check for app updates
- Reinstall app if necessary

### Sync Issues
- Ensure stable internet connection
- Check if server is operational
- Wait for automatic sync to complete
- Contact support if sync fails repeatedly

### Slow Performance
- Close other running apps
- Clear app cache
- Check device storage space
- Restart device
